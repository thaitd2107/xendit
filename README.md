## Description

Marvel character services.

## Installation

```bash
$ npm install
```

## Running the app
Configure .env file. Open and put public key, private key. You can register the keys at https://developer.marvel.com. 


```bash
# .env file
PUBLIC_KEY=Your key
PRIVATE_KEY=Your key
```

Start application:
```bash
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test

# test coverage
$ npm run test:cov
```
